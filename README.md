# Student Record System

A simple PHP project for managing student data.

## Files

- `main.php` - Entry point of the application.
- `operation.php` - Contains functions for data operations.

## Requirements

- PHP installed (compatible with version 7.4+).
- Web server (e.g., XAMPP) configured to serve the project.

## Features

- Add, edit and delete student records.
- View list of all students in a table.
- Search for students by name or ID.
- Simple form-based interface using PHP and HTML.

## Usage

1. Place the project folder in your web server's document root.
2. Access `main.php` via browser (e.g., http://localhost/student_data_system/main.php).
